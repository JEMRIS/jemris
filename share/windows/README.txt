To start: 

- Start MATLAB
- A new folder Simulation has been created as a sub directory 
  to your Documents folder; change into that directory
- Run jempath.m 
- Start GUIs acoordingly: 
  - JEMRIS_seq (Sequence design GUI)
  - JEMRIS_txrx (Coil array GUI)
  - JEMRIS_sim  (Simulation GUI)
